# MyKnowledge
随着信息技术的发展与阅读方式的多样化，纸质图书与电子图书并存，个人知识资源不断增长，传统以人工录入和简单分类为主的管理方式已难以满足高效与智能化需求。针对图书信息录入繁琐、资源分散及检索效率低等问题，本文设计并实现了一种基于网络爬虫与人工智能技术的个人知识管理系统。 系统采用前后端分离架构，基于Python Flask构建后端服务，结合Vue3实现前端交互，并通过MySQL或SQLite进行数据管理。功能上，通过ISBN扫码与网络爬虫实现图书信息的自动采集，降低人工成本；通过电子书扫描与元数据提取，实现纸质书与电子书的统一管理；引入自然语言处理技术，对图书内容进行语义分析，实现自动分类与标签生成；同时构建智能问答与推荐模块，提升系统的个性化服务能力。
